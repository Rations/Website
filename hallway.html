Hallway Problem [C]

Here's a fun problem that comes from the Feynman Lectures website.

You are an architect. Your client, living in Flatland, wants a building designed with a long 3 foot wide corridor which opens into a larger hallway. You must design the hallway for the minimum width w that will allow the inhabitants to move a 24 foot-long pole down the corridor and turn it into the hallway. The corridor is perpendicular to the hallway. Since this is Flatland, the pole cannot be tilted up. The pole is rigid. How wide must the hallway be?

This is the simplest form of an open problem known as the moving sofa problem. Fortunately, our version can be solved using the right geometric observations and elementary calculus.

Using similar triangles, let's solve for \(h\).

$$\frac{a}{x} = \frac{h}{24 - x}$$ $$h = \frac{a(24 - x)}{x}$$

From our drawing, we can interpret \(\theta\) as the independent variable on which all our lengths depend, so we first need to write our differentiable equation in terms of \(\theta\).

$$\tan\theta = \frac{a}{3}$$ $$a = 3\tan\theta$$ $$\cos\theta = \frac{3}{x}$$ $$x = 3\sec\theta$$ $$h = \frac{\tan\theta(24 - 3\sec\theta)}{\sec\theta}$$ $$h = 24\sin\theta - 3\tan\theta$$ $$\frac{dh}{d\theta} = 0 = 24\cos\theta - 3\sec^2\theta$$ $$\cos^3\theta = \frac{3}{24}$$ $$\theta = \frac{\pi}{3}$$

Now that we've found the optimal angle, we need to substitute it into our original equation to find the corresponding (minimum) \(h\):

$$h_\text{min} = 24\sin\frac{\pi}{3} - 3\tan\frac{\pi}{3}$$ $$h_\text{min} = 9\sqrt{3}$$

So as long as our hallway is wider than \(9\sqrt{3}\) m, we should be safe.

♛

The General Case

$$h = \frac{a(l - x)}{x}$$ $$h = \frac{\tan\theta(l - w\sec\theta)}{\sec\theta}$$ $$h = l\sin\theta - w\tan\theta$$ $$\frac{dh}{d\theta} = 0 = l\cos\theta - w\sec^2\theta$$ $$\theta = \arccos\sqrt[3]{\frac{w}{l}}$$ $$h_\text{min} = l\sin\arccos\sqrt[3]{\frac{w}{l}} - w\tan\arccos\sqrt[3]{\frac{w}{l}}$$

By examining a right triangle, we can see that \(\sin\arccos x = \sqrt{1 - x^2}\) and \(\tan\arccos x = \sqrt{\left(\frac{1}{x}\right)^2 - 1}\), we have:

$$h_\text{min} = l\sqrt{1 - \left(\frac{w}{l}\right)^{2/3}} - w\sqrt{\left(\frac{l}{w}\right)^{2/3} - 1}$$

\(w\) = width of starting hallway = m
\(l\) = pole length = m
\(h\) = width of ending hallway = m
